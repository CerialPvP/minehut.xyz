# Allowed File types

## OVERVIEW

Due to security reasons, you cannot upload certain file types to your servers.

## ALLOWED FILE TYPES

```text
.yml
.json
.txt
.sk
.nbt
.mcfunction
.schematic
.properties
.csv
.png
.midi
.nbs
.mcmeta
.dat
.conf
```
